task 1
Анализ социального графа пользователей

Описание проекта

Проект генерирует синтетическую социальную сеть пользователей с различными характеристиками, строит граф их связей и проводит графовый анализ.

Основные функции

1. Генерация пользователей

Функция generate_users(num_users=50) создает список пользователей со следующими атрибутами:

user_id: уникальный идентификатор
age: возраст от 16 до 65 лет
gender: пол ("male" или "female")
education_level: уровень образования ("high_school", "bachelor", "master")
messages_per_day: количество сообщений в день (0-100)
avg_message_length: средняя длина сообщения (5-200 символов)
positive_ratio: коэффициент позитивности сообщений (0.0-1.0)
is_active: активность пользователя (0 или 1)
friends: список друзей (заполняется позже)
2. Установка связей

Функция connect_users(users, avg_friends=5) устанавливает дружеские связи между пользователями:

Каждый пользователь получает случайное количество друзей
Обеспечивается взаимность дружбы
Максимальное количество друзей ограничено
3. Построение и визуализация графа

build_social_graph(users) создает граф с использованием библиотеки NetworkX
visualize_graph(graph) отображает граф с цветовой кодировкой:

Зеленый: активные пользователи
Красный: неактивные пользователи
4. Графовый анализ

Функция analyze_graph(graph) выполняет:

Анализ центральности по степени: определяет пользователей с наибольшим числом связей
Анализ посреднической центральности: находит пользователей, через которых проходит большинство кратчайших путей
Расчет кластерного коэффициента: измеряет степень склонности друзей пользователя дружить между собой
Выявление изолированных кластеров: находит группы пользователей, не связанных с основной сетью
**task  2 **
Описание проекта

Этот проект демонстрирует использование нейронной сети для классификации активности пользователей на основе их характеристик. Модель предсказывает, является ли пользователь активным (1) или неактивным (0), анализируя демографические данные и поведенческие паттерны.

Структура данных

Модель использует следующие признаки для предсказания:

Демографические признаки:

age (возраст) - числовой признак
gender (пол) - категориальный: male/female
education_level (уровень образования) - категориальный: high_school/bachelor/master
Поведенческие признаки:

messages_per_day (количество сообщений в день) - числовой
avg_message_length (средняя длина сообщения) - числовой
positive_ratio (доля позитивных сообщений) - числовой (0.0-1.0)
Целевая переменная:

is_active (активность пользователя) - бинарная: 0 (неактивен) или 1 (активен)
Архитектура модели

Проект использует нейронную сеть с следующей архитектурой:

Слой предобработки данных:

Стандартизация числовых признаков (StandardScaler)
One-Hot кодирование категориальных признаков
Нейронная сеть:

Входной слой: 8 нейронов (после предобработки)
Скрытый слой: 10 нейронов с активацией ReLU
Выходной слой: 1 нейрон с сигмоидной активацией для бинарной классификации
Технологический стек

Python 3.8+
NumPy - для числовых операций
Pandas - для обработки данных
Scikit-learn - для предобработки и оценки модели
TensorFlow/Keras - для построения нейронной сети
